Python Unit 1

Description:

In python unit 1 we learned the basics of programming in python. We first learned how to print statements. We then learned how to use basic variables, methods, and classes. We also covered strings and integers. In both the basics vaiables and Math Tricks lab we had to write a program that could take user unput and print the result of several calculations. This allowed us to learn how to use the basic operators to make the program carry out basic calculations. We then learned the basics about classes and objects. For example, in the Moving lab, we learned to use classes and methods to create an object that moves around the screen.

Concepts Learned:

  • Print Statements
  • Strings (str), Integers (int), float.
  • Input
  • Variables
  • Methods
  • Classes
  • Objects
  • Operators (+,-,%,*)

Python Unit 2

Description:

In Python Unit 2, we practiced the basics learned in unit 1. We learned about if statements and started using pygame. In the labrunner, we learned about how to make objects collide and prevent an image from moving off the screen. In the linerunner, we practiced the collide method and made an image hop to avoid obstacles. We also learned about how to keep score as the main object collided with rewards and obstacles.

Concepts Learned:
  • If statements
  • collide method
  • score variable
