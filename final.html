Final Project Content

In this project, we used all concepts learned throughout the class to create a game. For my game, I programmed a scratch version of the game runner that we did in python. I had a main object which could be moved up and down with the directional arrows. The goal was to avoid obstacles for as long as possible to get the highest score. The object only had three lives available. However, by collecting the rewards, the object could increase its number of lives.
