Java Content

Description:

The java unit was simply an introduction to java programming. We learned about the similarities and differences between java and python. we learned about classes, syntax, and print statements. We also learned about scanner imports, and methods. Finally, we went over strings and /n,/t, statements. We then practiced what we learned by creating a mini adventure story which consisted in using input from the player and print statements to guide the player through the story.
